# Instructions for Running and Viewing Results

## 1. Run HSPICE:
```
hspice iv_comparison.sp
```

## 2. Open WaveView:
```
wv &
```

## 3. Open the .tr0 file in WaveView

## 4. Plot the following Id-Vg curves:
- For actual current: `I(vd_mos)` vs `V(g_mos)` for various `V(d_mos)` values
- For predicted current: `I(G_NN)` vs `V(g_nn)` for the same `V(d_nn)` values

## 5. Plot the following Id-Vd curves:
- For actual current: `I(vd_mos)` vs `V(d_mos)` for various `V(g_mos)` values
- For predicted current: `I(G_NN)` vs `V(d_nn)` for the same `V(g_nn)` values

## 6. Tips for better visualization:
- Use log scale for current (right-click on Y-axis → Scale → Log)
- Use different colors for actual and predicted curves
- Add grid lines (View → Grid)
- Add curve labels (right-click on curve → Add Curve Label)
